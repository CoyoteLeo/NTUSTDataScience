# environment
python 3.6+
```bash
# create virtual python environment 
$ python3 -m venv venv
$ .\venv\Scripts\activate
$ pip install -r requirements.txt
```

if you pip install any module, please update the requirements.txt via the command below
```bash
$ pip freeze > requirements.txt
```
